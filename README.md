## 📖 About the Project

This project was built as an extension of my professional portfolio.

Instead of storing certificates as individual PDF files, this website presents them in a clean, organized and responsive interface where recruiters and employers can easily explore my completed courses and technical skills.

## ✨ Features

- Professional landing page
- Responsive Design
- Modern UI
- Certification gallery
- Learning Journey Timeline
- Technical Skills section
